# 多风格图像风格迁移工具：一步一步跟做手册

> 这份文档不是概念介绍，而是**从空文件夹开始，一步一步做出可运行项目**的操作手册。  
> 你当前配置：`16GB 内存 + RTX 4060 8GB 显存`。  
> 推荐路线：先做 `CycleGAN 单风格 MVP`，再扩展到 `10+ 风格`。  
> 第一目标：先让 `普通照片 -> 梵高风格` 跑通训练、推理和 Gradio 页面。  

---

## 0. 使用这份手册的方式

### 0.1 先记住一个原则

不要一开始就做 10 个风格。  
正确顺序是：

```text
第 1 个风格跑通完整闭环
-> 再复制配置训练第 2 个风格
-> 再做多风格下拉选择
-> 再做批量处理、高清导出、视频处理
```

如果你一开始就同时做 10 个风格，会遇到这些问题：

- 不知道是数据问题、模型问题还是界面问题。
- 训练时间太长，排错困难。
- 每种风格参数可能都要微调。
- Gradio 界面没有模型可用时也无法验证。

所以本手册先让你完成：

```text
photo -> vangogh
```

之后再扩展：

```text
photo -> picasso
photo -> ink
photo -> anime
...
```

### 0.2 本手册默认使用 Windows PowerShell

所有命令默认在 PowerShell 中执行。  
如果命令中出现 `^`，表示 Windows 多行命令换行。

你也可以直接写成一行执行。

### 0.3 每一步都有验收标准

每一步都按这个格式：

```text
目标
操作
产物
验收
失败排查
```

你照着做时，不要跳过“验收”。  
如果某一步没验收通过，先不要继续下一步。

## 3. 第 0 阶段：检查电脑环境

### 3.1 目标

确认你的电脑能用 NVIDIA 显卡跑 PyTorch。

你的配置：

```text
内存：16GB
显卡：RTX 4060 8GB
```

这个配置适合：

- `256x256` CycleGAN 训练
- `batch_size=1`
- 单张图片推理

不适合第一版直接做：

- `1024x1024` 训练
- 大 batch 训练
- 同时加载 10 个模型

## 6. 第 3 阶段：准备数据集

### 6.1 目标

准备两类图片：

```text
A 域：普通照片 photo
B 域：梵高风格图片 vangogh
```

CycleGAN 不需要一一对应图片。  
你只需要两堆图片。

### 6.2 数据放置位置

普通照片放这里：

```text
data/raw/photo/
```

梵高风格图放这里：

```text
data/raw/vangogh/
```

### 6.3 第一轮最小数据量

只为了测试代码能跑：

```text
photo: 50 张
vangogh: 50 张
```

用于初步训练：

```text
photo: 500 张
vangogh: 500 张
```

用于展示效果：

```text
photo: 1500 张以上
vangogh: 1500 张以上
```

### 6.4 图片要求

建议：

- 图片宽高都大于 `256`
- 格式为 `jpg`、`jpeg`、`png`
- 尽量 RGB 图片
- 不要大量重复图
- 不要混入水印明显的图
- 不要混入纯色图

### 6.5 普通照片应该包含什么

`photo` 域尽量丰富：

```text
人像
风景
城市
建筑
动物
街景
室内
植物
静物
```

如果只放风景图，模型处理人像时可能很差。

### 6.6 梵高风格图应该包含什么

`vangogh` 域尽量风格统一：

```text
梵高画作
类似梵高笔触的油画
高质量艺术图
```

不要混入：

```text
真实照片
动漫图
水墨画
低清截图
大量边框或文字
```

### 6.7 文件命名建议

建议重命名为：

```text
000001.jpg
000002.jpg
000003.jpg
```

不要使用：

```text
中文文件名
空格
括号
特殊符号
超长文件名
```

这样可以减少 OpenCV、PIL、命令行路径问题。

### 6.8 当前阶段验收

执行：

```powershell
(Get-ChildItem data\raw\photo -File).Count
(Get-ChildItem data\raw\vangogh -File).Count
```

你应该看到两个数字。  
第一轮测试至少：

```text

50
```

正式训练建议至少：

```text
500
500
```

```text

```

运行方式设计为：

```powershell

```

### 7.3 创建清洗脚本

创建文件：

```text
datasets/preprocess.py
```

脚本参数设计：

```powershell
python datasets/preprocess.py ^
  --input data\raw\photo ^
  --output data\clean\photo ^
  --min-size 256
```

```powershell
python datasets/preprocess.py ^
  --input data\raw\vangogh ^
  --output data\clean\vangogh ^
  --min-size 256
```

### 7.4 清洗脚本应该做什么

每张图片：

```text
1. 用 PIL.Image.open 打开
2. img.verify 或正常 load
3. 转成 RGB
4. 判断 width >= min_size 且 height >= min_size
5. 保存成 jpg
6. 文件名改成 000001.jpg 这种格式
```

### 7.5 清洗后目录


## 10. 第 7 阶段：实现数据加载器

### 10.1 目标

实现 CycleGAN 的非配对数据加载。

### 10.2 CycleGAN 数据加载逻辑

每次训练取：

```text
一张普通照片 A
一张随机梵高图片 B
```

注意：  
A 和 B 不需要对应。

### 10.3 创建数据集文件

创建文件：

```text
datasets/unpaired_dataset.py
```

里面应该包含：

```python
class UnpairedImageDataset(torch.utils.data.Dataset):
    def __init__(self, dir_a, dir_b, resize_size, image_size):
        ...

    def __len__(self):
        ...

    def __getitem__(self, index):
        ...
```

### 10.4 `__getitem__` 应该返回什么

返回字典：

```python
{
    "A": image_a_tensor,
    "B": image_b_tensor,
    "A_path": path_a,
    "B_path": path_b,
}
```

### 10.5 图像变换流程

训练阶段：

```text
读取图片
转 RGB
resize 到 286
随机裁剪到 256
随机水平翻转
转 tensor
归一化到 [-1, 1]
```

测试阶段：

```text
读取图片
转 RGB
resize 到 256
中心裁剪或直接缩放
转 tensor
归一化到 [-1, 1]
```

### 10.6 为什么归一化到 [-1, 1]

生成器最后通常用：

```python
nn.Tanh()
```

`Tanh` 输出范围是：

```text
[-1, 1]
```

所以训练输入也要归一化到这个范围。

### 10.7 当前阶段验收

创建一个测试脚本或在 Python 交互中检查：

```text
dataset[0]["A"].shape == [3, 256, 256]
dataset[0]["B"].shape == [3, 256, 256]
```

如果能拿到 tensor，说明数据加载器基本成功。

## 11. 第 8 阶段：实现模型结构

### 11.1 目标

实现 CycleGAN 需要的网络：

```text
G_AB：照片 -> 梵高
G_BA：梵高 -> 照片
D_A：判断是否像照片
D_B：判断是否像梵高
```

### 11.2 创建模型文件

创建这些文件：

```text
models/blocks.py
models/generator.py
models/discriminator.py
models/losses.py
models/cyclegan.py
```

### 11.3 blocks.py 应该包含什么

`blocks.py` 放基础模块：

```text
ResidualBlock
weights_init_normal
```

`ResidualBlock` 结构：

```text
ReflectionPad2d
Conv2d
InstanceNorm2d
ReLU
ReflectionPad2d
Conv2d
InstanceNorm2d
skip connection
```

### 11.4 generator.py 应该包含什么

`generator.py` 放：

```python
class ResnetGenerator(nn.Module):
    ...
```

推荐结构：

```text
输入 3 通道
-> 7x7 卷积
-> 下采样 2 次
-> 6 个 ResidualBlock
-> 上采样 2 次
-> 7x7 卷积
-> Tanh
```

你的显卡推荐参数：

```text
ngf = 64
n_blocks = 6
image_size = 256
batch_size = 1
```

如果爆显存：

```text
ngf = 32
n_blocks = 6
```

### 11.5 discriminator.py 应该包含什么

`discriminator.py` 放：

```python
class PatchDiscriminator(nn.Module):
    ...
```

推荐结构：

```text
Conv2d
LeakyReLU
Conv2d
InstanceNorm2d
LeakyReLU
Conv2d
InstanceNorm2d
LeakyReLU
Conv2d 输出 patch 判断图
```

PatchGAN 不输出单个真假值，而是输出一张 patch map。

### 11.6 losses.py 应该包含什么

`losses.py` 放：

```text
GANLoss
ImagePool
```

损失函数：

```text
adversarial loss
cycle consistency loss

```

```python
import torch.nn as nn


class PatchDiscriminator(nn.Module):
    def __init__(self, input_nc=3, ndf=64):
        super().__init__()

        layers = [
            nn.Conv2d(input_nc, ndf, kernel_size=4, stride=2, padding=1),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Conv2d(ndf, ndf * 2, kernel_size=4, stride=2, padding=1),
            nn.InstanceNorm2d(ndf * 2),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Conv2d(ndf * 2, ndf * 4, kernel_size=4, stride=2, padding=1),
            nn.InstanceNorm2d(ndf * 4),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Conv2d(ndf * 4, ndf * 8, kernel_size=4, stride=1, padding=1),
            nn.InstanceNorm2d(ndf * 8),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Conv2d(ndf * 8, 1, kernel_size=4, stride=1, padding=1),
        ]

        self.model = nn.Sequential(*layers)

    def forward(self, x):
        return self.model(x)
```

```python
import random

import torch
import torch.nn as nn


class GANLoss(nn.Module):
    def __init__(self):
        super().__init__()
        self.loss = nn.MSELoss()

    def forward(self, prediction, target_is_real):
        target = torch.ones_like(prediction) if target_is_real else torch.zeros_like(prediction)
        return self.loss(prediction, target)


class ImagePool:
    def __init__(self, pool_size=50):
        self.pool_size = pool_size
        self.images = []

    def query(self, images):
        if self.pool_size == 0:
            return images

        return_images = []
        for image in images:
            image = image.unsqueeze(0)
            if len(self.images) < self.pool_size:
                self.images.append(image)
                return_images.append(image)
            else:
                if random.random() > 0.5:
                    idx = random.randint(0, self.pool_size - 1)
                    old_image = self.images[idx].clone()
                    self.images[idx] = image
                    return_images.append(old_image)
                else:
                    return_images.append(image)

        return torch.cat(return_images, dim=0)
```

## 37. 代码模板：models/cyclegan.py

说明：这份模板先使用普通 FP32 训练。  
先跑通后，再加 AMP 混合精度会更稳。

代码：

```python
import torch
import torch.nn as nn

from models.blocks import init_weights
from models.discriminator import PatchDiscriminator
from models.generator import ResnetGenerator
from models.losses import GANLoss, ImagePool


def set_requires_grad(nets, requires_grad=False):
    if not isinstance(nets, list):
        nets = [nets]
    for net in nets:
        for param in net.parameters():
            param.requires_grad = requires_grad


class CycleGANModel:
    def __init__(self, config, device):
        self.config = config
        self.device = device

        model_cfg = config["model"]
        train_cfg = config["train"]

        self.G_AB = ResnetGenerator(3, 3, model_cfg["ngf"], model_cfg["n_blocks"]).to(device)
        self.G_BA = ResnetGenerator(3, 3, model_cfg["ngf"], model_cfg["n_blocks"]).to(device)
        self.D_A = PatchDiscriminator(3, model_cfg["ndf"]).to(device)
        self.D_B = PatchDiscriminator(3, model_cfg["ndf"]).to(device)

        self.G_AB.apply(init_weights)
        self.G_BA.apply(init_weights)
        self.D_A.apply(init_weights)
        self.D_B.apply(init_weights)

        self.criterion_gan = GANLoss().to(device)
        self.criterion_cycle = nn.L1Loss()
        self.criterion_identity = nn.L1Loss()

        self.lambda_cycle = train_cfg["lambda_cycle"]
        self.lambda_identity = train_cfg["lambda_identity"]

        lr = train_cfg["lr"]
        beta1 = train_cfg["beta1"]

        self.optimizer_G = torch.optim.Adam(
            list(self.G_AB.parameters()) + list(self.G_BA.parameters()),
            lr=lr,
            betas=(beta1, 0.999),
        )
        self.optimizer_D_A = torch.optim.Adam(self.D_A.parameters(), lr=lr, betas=(beta1, 0.999))
        self.optimizer_D_B = torch.optim.Adam(self.D_B.parameters(), lr=lr, betas=(beta1, 0.999))

        self.fake_A_pool = ImagePool()
        self.fake_B_pool = ImagePool()
        self.losses = {}

    def set_input(self, batch):
        self.real_A = batch["A"].to(self.device)
        self.real_B = batch["B"].to(self.device)

    def forward(self):
        self.fake_B = self.G_AB(self.real_A)
        self.rec_A = self.G_BA(self.fake_B)
        self.fake_A = self.G_BA(self.real_B)
        self.rec_B = self.G_AB(self.fake_A)

    def backward_G(self):
        idt_B = self.G_AB(self.real_B)
        idt_A = self.G_BA(self.real_A)

        loss_idt_B = self.criterion_identity(idt_B, self.real_B) * self.lambda_cycle * self.lambda_identity
        loss_idt_A = self.criterion_identity(idt_A, self.real_A) * self.lambda_cycle * self.lambda_identity

        loss_G_AB = self.criterion_gan(self.D_B(self.fake_B), True)
        loss_G_BA = self.criterion_gan(self.D_A(self.fake_A), True)

        loss_cycle_A = self.criterion_cycle(self.rec_A, self.real_A) * self.lambda_cycle
        loss_cycle_B = self.criterion_cycle(self.rec_B, self.real_B) * self.lambda_cycle

        loss_G = loss_G_AB + loss_G_BA + loss_cycle_A + loss_cycle_B + loss_idt_A + loss_idt_B
        loss_G.backward()

        self.losses.update({
            "loss_G": float(loss_G.item()),
            "loss_G_AB": float(loss_G_AB.item()),
            "loss_G_BA": float(loss_G_BA.item()),
            "loss_cycle_A": float(loss_cycle_A.item()),
            "loss_cycle_B": float(loss_cycle_B.item()),
            "loss_idt_A": float(loss_idt_A.item()),
            "loss_idt_B": float(loss_idt_B.item()),
        })

    def backward_D_basic(self, net_d, real, fake):
        pred_real = net_d(real)
        loss_real = self.criterion_gan(pred_real, True)

        pred_fake = net_d(fake.detach())
        loss_fake = self.criterion_gan(pred_fake, False)

        loss_d = (loss_real + loss_fake) * 0.5
        loss_d.backward()
        return loss_d

    def backward_D_A(self):
        fake_A = self.fake_A_pool.query(self.fake_A.detach())
        loss_D_A = self.backward_D_basic(self.D_A, self.real_A, fake_A)
        self.losses["loss_D_A"] = float(loss_D_A.item())

    def backward_D_B(self):
        fake_B = self.fake_B_pool.query(self.fake_B.detach())
        loss_D_B = self.backward_D_basic(self.D_B, self.real_B, fake_B)
        self.losses["loss_D_B"] = float(loss_D_B.item())

    def optimize_parameters(self):
        self.forward()

        set_requires_grad([self.D_A, self.D_B], False)
        self.optimizer_G.zero_grad(set_to_none=True)
        self.backward_G()
        self.optimizer_G.step()

        set_requires_grad([self.D_A, self.D_B], True)
        self.optimizer_D_A.zero_grad(set_to_none=True)
        self.backward_D_A()
        self.optimizer_D_A.step()

        self.optimizer_D_B.zero_grad(set_to_none=True)
        self.backward_D_B()
        self.optimizer_D_B.step()

    def save(self, path, epoch):
        torch.save({
            "epoch": epoch,
            "G_AB": self.G_AB.state_dict(),
            "G_BA": self.G_BA.state_dict(),
            "D_A": self.D_A.state_dict(),
            "D_B": self.D_B.state_dict(),
            "optimizer_G": self.optimizer_G.state_dict(),
            "optimizer_D_A": self.optimizer_D_A.state_dict(),
            "optimizer_D_B": self.optimizer_D_B.state_dict(),
        }, path)

    def save_generator_ab(self, path):
        torch.save(self.G_AB.state_dict(), path)

    def load(self, path):
        checkpoint = torch.load(path, map_location=self.device)
        self.G_AB.load_state_dict(checkpoint["G_AB"])
        self.G_BA.load_state_dict(checkpoint["G_BA"])
        self.D_A.load_state_dict(checkpoint["D_A"])
        self.D_B.load_state_dict(checkpoint["D_B"])
        self.optimizer_G.load_state_dict(checkpoint["optimizer_G"])
        self.optimizer_D_A.load_state_dict(checkpoint["optimizer_D_A"])
        self.optimizer_D_B.load_state_dict(checkpoint["optimizer_D_B"])
        return checkpoint.get("epoch", 0)
```

代码：

```python
import argparse
from pathlib import Path

import torch
from torch.utils.data import DataLoader
from tqdm import tqdm

from datasets.unpaired_dataset import UnpairedImageDataset
from models.cyclegan import CycleGANModel
from training.train_utils import append_loss_csv, ensure_dir, load_config, save_sample_grid, set_seed


def parse_args():
    parser = argparse.ArgumentParser()
    parser.add_argument("--config", required=True)
    parser.add_argument("--epochs", type=int, default=None)
    parser.add_argument("--resume", default=None)
    return parser.parse_args()


def main():
    args = parse_args()
    config = load_config(args.config)

    if args.epochs is not None:
        config["train"]["epochs"] = args.epochs

    set_seed(config["seed"])

    device = torch.device("cuda" if torch.cuda.is_available() and config["device"] == "cuda" else "cpu")
    print(f"device: {device}")

    checkpoint_dir = Path(config["output"]["checkpoint_dir"])
    log_dir = Path(config["output"]["log_dir"])
    sample_dir = log_dir / "samples"
    ensure_dir(checkpoint_dir)
    ensure_dir(sample_dir)

    train_cfg = config["train"]
    dataset_cfg = config["dataset"]

    dataset = UnpairedImageDataset(
        dataset_cfg["trainA"],
        dataset_cfg["trainB"],
        resize_size=train_cfg["resize_size"],
        image_size=train_cfg["image_size"],
        train=True,
    )
    dataloader = DataLoader(
        dataset,
        batch_size=train_cfg["batch_size"],
        shuffle=True,
        num_workers=train_cfg["num_workers"],
        pin_memory=True,
    )

    model = CycleGANModel(config, device)
    start_epoch = 1

    if args.resume:
        last_epoch = model.load(args.resume)
        start_epoch = last_epoch + 1
        print(f"resume from epoch {last_epoch}")

    total_epochs = train_cfg["epochs"]

    for epoch in range(start_epoch, total_epochs + 1):
        progress = tqdm(dataloader, desc=f"epoch {epoch}/{total_epochs}")

        for batch in progress:
            model.set_input(batch)
            model.optimize_parameters()
            progress.set_postfix({
                "G": f"{model.losses.get('loss_G', 0):.3f}",
                "D_A": f"{model.losses.get('loss_D_A', 0):.3f}",
                "D_B": f"{model.losses.get('loss_D_B', 0):.3f}",
            })

        row = {"epoch": epoch}
        row.update(model.losses)
        append_loss_csv(log_dir / "losses.csv", row)

        if epoch % train_cfg["sample_freq"] == 0:
            save_sample_grid(
                sample_dir / f"epoch_{epoch:03d}.jpg",
                model.real_A,
                model.fake_B,
                model.rec_A,
                model.real_B,
                model.fake_A,
                model.rec_B,
            )

        if epoch % train_cfg["save_freq"] == 0 or epoch == total_epochs:
            model.save(checkpoint_dir / f"epoch_{epoch:03d}.pth", epoch)

        model.save(checkpoint_dir / "latest.pth", epoch)
        model.save_generator_ab(checkpoint_dir / "latest_G_AB.pth")

    print("training finished")


if __name__ == "__main__":
    main()
```

```python
def load_generator(style_key, device):
    style = STYLE_REGISTRY[style_key]
    model = ResnetGenerator(3, 3, ngf=style["ngf"], n_blocks=style["n_blocks"]).to(device)
    state = torch.load(style["checkpoint"], map_location=device)
    model.load_state_dict(state)
    model.eval()
    return model


def tensor_to_pil(tensor):
    tensor = tensor.detach().cpu().squeeze(0)
    tensor = ((tensor + 1) / 2).clamp(0, 1)
    return transforms.ToPILImage()(tensor)


def stylize_pil(image, model, image_size, device):
    original_size = image.size
    transform = transforms.Compose([
        transforms.Resize((image_size, image_size)),
        transforms.ToTensor(),
        transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5)),
    ])

    x = transform(image.convert("RGB")).unsqueeze(0).to(device)
    with torch.no_grad():
        y = model(x)

    output = tensor_to_pil(y)
    return output.resize(original_size, Image.BICUBIC)


def infer_image(input_path, style_key, output_path):
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    style = STYLE_REGISTRY[style_key]
    model = load_generator(style_key, device)

    image = Image.open(input_path).convert("RGB")
    output = stylize_pil(image, model, style["image_size"], device)

    output_path = Path(output_path)
    output_path.parent.mkdir(parents=True, exist_ok=True)
    output.save(output_path, quality=95)
    print(f"saved: {output_path}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--input", required=True)
    parser.add_argument("--style", required=True, choices=list(STYLE_REGISTRY.keys()))
    parser.add_argument("--output", required=True)
    args = parser.parse_args()
    infer_image(args.input, args.style, args.output)


if __name__ == "__main__":
```

```python
def batch_infer(input_dir, style_key, output_dir):
    input_dir = Path(input_dir)
    output_dir = Path(output_dir)

    model = load_generator(style_key, device)

    log_path = output_dir / "process_log.csv"
    with open(log_path, "w", newline="", encoding="utf-8") as f:
        writer = csv.DictWriter(f, fieldnames=["filename", "status", "message", "output_path", "time_ms"])
        writer.writeheader()

        for file in files:
            start = time.time()
            try:
                image = Image.open(file).convert("RGB")
                output = stylize_pil(image, model, style["image_size"], device)
                output_path = output_dir / f"{file.stem}_{style_key}.jpg"
                output.save(output_path, quality=95)
                status = "ok"
                message = ""
            except Exception as exc:
                output_path = ""
                status = "failed"
                message = str(exc)

            time_ms = int((time.time() - start) * 1000)
            writer.writerow({
                "filename": file.name,
                "status": status,
                "message": message,
                "output_path": str(output_path),
                "time_ms": time_ms,
            })
            print(f"{file.name}: {status}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--input-dir", required=True)
    parser.add_argument("--style", required=True, choices=list(STYLE_REGISTRY.keys()))
    parser.add_argument("--output-dir", required=True)
    args = parser.parse_args()
    batch_infer(args.input_dir, args.style, args.output_dir)


if __name__ == "__main__":
    main()
```

## 44. 这些代码模板的限制

这套模板是为了先跑通项目，所以有这些限制：


公式：

## 29. 代码模板：datasets/preprocess.py

目标：清洗原始图片，过滤坏图和过小图片，并统一保存为 jpg。

文件路径：

```text
datasets/preprocess.py
```

代码：

```python
import argparse
from pathlib import Path
from PIL import Image


IMAGE_EXTS = {".jpg", ".jpeg", ".png", ".webp", ".bmp"}


def preprocess_images(input_dir: str, output_dir: str, min_size: int) -> None:
    input_path = Path(input_dir)
    output_path = Path(output_dir)
    output_path.mkdir(parents=True, exist_ok=True)

    files = [p for p in input_path.rglob("*") if p.suffix.lower() in IMAGE_EXTS]
    ok_count = 0
    skip_count = 0

    for file in files:
        try:
            with Image.open(file) as img:
                img = img.convert("RGB")
                w, h = img.size
                if w < min_size or h < min_size:
                    skip_count += 1
                    continue

                ok_count += 1
                img.save(output_path / f"{ok_count:06d}.jpg", quality=95)
        except Exception as exc:
            skip_count += 1
            print(f"skip {file}: {exc}")

    print(f"saved: {ok_count}")
    print(f"skipped: {skip_count}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--input", required=True)
    parser.add_argument("--output", required=True)
    parser.add_argument("--min-size", type=int, default=256)
    args = parser.parse_args()
    preprocess_images(args.input, args.output, args.min_size)


if __name__ == "__main__":
    main()
```

## 30. 代码模板：datasets/split_dataset.py

目标：把清洗后的图片拆成 `trainA/trainB/testA/testB`。

文件路径：

```text
datasets/split_dataset.py
```

代码：

```python
import argparse
import random
import shutil
from pathlib import Path


IMAGE_EXTS = {".jpg", ".jpeg", ".png", ".webp", ".bmp"}


def list_images(path: Path):
    return [p for p in path.iterdir() if p.is_file() and p.suffix.lower() in IMAGE_EXTS]


def copy_split(files, train_dir: Path, test_dir: Path, test_ratio: float):
    train_dir.mkdir(parents=True, exist_ok=True)
    test_dir.mkdir(parents=True, exist_ok=True)

    random.shuffle(files)
    test_count = max(1, int(len(files) * test_ratio))
    test_files = files[:test_count]
    train_files = files[test_count:]

    for idx, file in enumerate(train_files, start=1):
        shutil.copy2(file, train_dir / f"{idx:06d}{file.suffix.lower()}")

    for idx, file in enumerate(test_files, start=1):
        shutil.copy2(file, test_dir / f"{idx:06d}{file.suffix.lower()}")

    return len(train_files), len(test_files)


def split_dataset(photo_dir, style_dir, output_dir, test_ratio, seed):
    random.seed(seed)
    output = Path(output_dir)

    photo_files = list_images(Path(photo_dir))
    style_files = list_images(Path(style_dir))

    if not photo_files:
        raise RuntimeError(f"no photo images found: {photo_dir}")
    if not style_files:
        raise RuntimeError(f"no style images found: {style_dir}")

    train_a, test_a = copy_split(photo_files, output / "trainA", output / "testA", test_ratio)
    train_b, test_b = copy_split(style_files, output / "trainB", output / "testB", test_ratio)

    print(f"trainA: {train_a}, testA: {test_a}")
    print(f"trainB: {train_b}, testB: {test_b}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--photo", required=True)
    parser.add_argument("--style", required=True)
    parser.add_argument("--output", required=True)
    parser.add_argument("--test-ratio", type=float, default=0.1)
    parser.add_argument("--seed", type=int, default=42)
    args = parser.parse_args()
    split_dataset(args.photo, args.style, args.output, args.test_ratio, args.seed)


if __name__ == "__main__":
    main()
```

运行：

```powershell
python datasets/split_dataset.py --photo data\clean\photo --style data\clean\vangogh --output data\prepared\vangogh --test-ratio 0.1
```

## 31. 代码模板：datasets/unpaired_dataset.py

目标：训练时随机读取一张普通照片和一张风格图。

文件路径：

```text
datasets/unpaired_dataset.py
```

代码：

```python
import random
from pathlib import Path

from PIL import Image
from torch.utils.data import Dataset
from torchvision import transforms


IMAGE_EXTS = {".jpg", ".jpeg", ".png", ".webp", ".bmp"}


class UnpairedImageDataset(Dataset):
    def __init__(self, dir_a, dir_b, resize_size=286, image_size=256, train=True):
        self.files_a = sorted([p for p in Path(dir_a).iterdir() if p.suffix.lower() in IMAGE_EXTS])
        self.files_b = sorted([p for p in Path(dir_b).iterdir() if p.suffix.lower() in IMAGE_EXTS])

        if not self.files_a:
            raise RuntimeError(f"no images in {dir_a}")
        if not self.files_b:
            raise RuntimeError(f"no images in {dir_b}")

        if train:
            self.transform = transforms.Compose([
                transforms.Resize((resize_size, resize_size)),
                transforms.RandomCrop(image_size),
                transforms.RandomHorizontalFlip(),
                transforms.ToTensor(),
                transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5)),
            ])
        else:
            self.transform = transforms.Compose([
                transforms.Resize((image_size, image_size)),
                transforms.ToTensor(),
                transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5)),
            ])

    def __len__(self):
        return max(len(self.files_a), len(self.files_b))

    def __getitem__(self, index):
        path_a = self.files_a[index % len(self.files_a)]
        path_b = random.choice(self.files_b)

        image_a = Image.open(path_a).convert("RGB")
        image_b = Image.open(path_b).convert("RGB")

        return {
            "A": self.transform(image_a),
            "B": self.transform(image_b),
            "A_path": str(path_a),
            "B_path": str(path_b),
        }
```

文件路径：

```python
import torch.nn as nn


class ResidualBlock(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.block = nn.Sequential(
            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, kernel_size=3),
            nn.InstanceNorm2d(channels),
            nn.ReLU(inplace=True),
            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, kernel_size=3),
            nn.InstanceNorm2d(channels),
        )

    def forward(self, x):
        return x + self.block(x)


def init_weights(module):
    classname = module.__class__.__name__
    if classname.find("Conv") != -1:
        nn.init.normal_(module.weight.data, 0.0, 0.02)
        if getattr(module, "bias", None) is not None:
            nn.init.constant_(module.bias.data, 0.0)
```

## 33. 代码模板：models/generator.py

目标：实现 `photo -> style` 的生成器。

文件路径：

```text

```

代码：

```python
import torch.nn as nn

from models.blocks import ResidualBlock


class ResnetGenerator(nn.Module):
    def __init__(self, input_nc=3, output_nc=3, ngf=64, n_blocks=6):
        super().__init__()

        layers = [
            nn.ReflectionPad2d(3),
            nn.Conv2d(input_nc, ngf, kernel_size=7),
            nn.InstanceNorm2d(ngf),
            nn.ReLU(inplace=True),
        ]

        in_channels = ngf
        out_channels = ngf * 2
        for _ in range(2):
            layers += [
                nn.Conv2d(in_channels, out_channels, kernel_size=3, stride=2, padding=1),
                nn.InstanceNorm2d(out_channels),
                nn.ReLU(inplace=True),
            ]
            in_channels = out_channels
            out_channels *= 2

        for _ in range(n_blocks):
            layers.append(ResidualBlock(in_channels))

        out_channels = in_channels // 2
        for _ in range(2):
            layers += [
                nn.ConvTranspose2d(
                    in_channels,
                    out_channels,
                    kernel_size=3,
                    stride=2,
                    padding=1,
                    output_padding=1,
                ),
                nn.InstanceNorm2d(out_channels),
                nn.ReLU(inplace=True),
            ]
            in_channels = out_channels
            out_channels //= 2

        layers += [
            nn.ReflectionPad2d(3),
            nn.Conv2d(in_channels, output_nc, kernel_size=7),
            nn.Tanh(),
        ]

        self.model = nn.Sequential(*layers)

    def forward(self, x):
        return self.model(x)
```
